# Roomies — A Bunk-Up

A configurable bed and crib assignment game for the family to decide who rooms with whom. Set up your houses, bedrooms, and furniture; add your kids and au pairs; then drag everyone into their spots and settle the bunk-up once and for all.

## Play

Open `index.html` in a browser, or visit [roomies.kibera.com](https://roomies.kibera.com).

## Tech

- Single `index.html` — no build step
- Vanilla JavaScript
- canvas-confetti via CDN (for the celebration screen)
- Cormorant Garamond + Source Serif 4 (Google Fonts)

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `roomies`).
2. Push this folder to `main`.
3. In the repo settings → Pages, set source to **main branch / root**.
4. Add a custom domain pointing to `roomies.kibera.com`.
5. Hard-refresh (`Cmd+Shift+R`) after deploy to clear the cache.
